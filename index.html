Project Case Study

Pagtakhan Medical Clinic – Online Appointment & Queue System

Undergraduate Thesis Project | Solo Developer

Project Context

This system was developed as my undergraduate thesis project, addressing real-world operational challenges faced by a local medical clinic in managing patient appointments and queues.

Problem

The clinic relied on manual scheduling, resulting in inefficient patient flow, long waiting times, and limited visibility of queue status.

Solution

A web-based appointment and queueing system that allows patients to book schedules online while enabling administrators to manage queues dynamically and efficiently.

My Role

Sole developer responsible for system design, backend logic, database architecture, frontend integration, and deployment.

Technologies

Django, Python, PostgreSQL, HTML, CSS, Git, Docker, Heroku

Key Features
  • Online appointment scheduling
  • Automated queue number assignment
  • Doctor availability and time-slot management
  • Admin-controlled queue progression
  • Status-based appointment tracking
Outcome

The system successfully digitized clinic operations, improved queue efficiency, and demonstrated the practical application of full-stack web development concepts in a real-world environment.
